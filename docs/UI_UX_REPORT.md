# UI_UX_REPORT.md — Murad Learning Planner
**Date:** 2026-07-21

## فلسفة التنفيذ
التصميم القائم (Glassmorphism داكن/فاتح بمستوى SaaS) **هو هوية المنصة** — التزمنا بتحسينه لا استبداله. كل الإضافات طبقات additive قابلة للإزالة، ولا تغيير على بنية أي صفحة أو مكوّن.

## ما أُضيف

### 1. المقدمة السينمائية (`css/intro.css` + `js/intro.js`)
تجربة افتتاحية وفق المشهد المطلوب:
- فضاء عميق (110 نجمة متلألئة) → **بنية عصبية** (26 عقدة + وصلات synapse ديناميكية) → نواة مضيئة → تقارب 90 جسيماً → **وميض تحوّل** → تجمّع الشعار واسم المنصة → انتقال ناعم للتطبيق (~4.2s).
- **قابلة للتخطي**: زر «تخطّي» + Esc/Enter/Space + النقر في أي مكان.
- **إمكانية الوصول**: تُتجاوز كلياً مع `prefers-reduced-motion`، ولها toggle في الإعدادات («المقدمة السينمائية عند البدء»)، وتعمل مرة واحدة لكل جلسة.
- **الأداء**: Canvas 2D واحد، rAF واحد، dpr≤2، صفر تكلفة بعد الانتهاء (إزالة كاملة من الـ DOM). لا مكتبات 3D — «3D باعتدال» تحقق عبر العمق البصري (توهج شعاعي، parallax جسيمات، منظور بيضاوي).

### 2. طبقة الصقل (`css/polish.css`)
- **Page transitions**: دخول ناعم (rise + fade, 380ms cubic-bezier).
- **Micro-interactions**: أزرار برفعة spring خفيفة عند hover وضغطة scale عند active؛ بطاقات بعمق ظل عند hover؛ nav-items بانزياح اتجاهي (RTL-aware).
- **Focus states**: توهج حلقي موحد لكل الحقول + إطار داخلي للخلايا القابلة للتحرير.
- **Skeleton shimmer** utility للمحتوى غير المتزامن.
- **Toast/Modal** بحركة pop مضبوطة.
- كل ما سبق يُعطَّل بالكامل تحت `prefers-reduced-motion`.

### 3. الوصولية (39 → 0)
- `aria-label` + `title` لكل زر أيقوني (تنقّل أسابيع، إغلاق اللوحات، أزرار الصوت، أزرار المزودات).
- ربط `for=` بين كل label وحقله (24 حقلاً).
- Skip-to-content link (يظهر عند التركيز فقط، بتدرج هوية المنصة).
- `aria-hidden` للأيقونات الزخرفية.

### 4. SEO / Head
- Title وصفي ثنائي اللغة، meta description، Open Graph، `theme-color` لكل وضع، `color-scheme`، `preconnect` لـ jsDelivr.

## ما لم يُغيَّر عمداً
- ألوان وتدرجات وهوية المنصة، بنية Sidebar/Navbar/القوائم، تخطيط كل الصفحات، وواجهة AI Workspace القائمة (v5) — كانت بالفعل بمستوى عالٍ وتعمل بلا أخطاء؛ التدخل فيها مخاطرة بلا مكسب.
- «الصفحة الرئيسية التسويقية» (Hero/Pricing/Testimonials): المنصة **أداة إنتاجية شخصية** لا موقع تسويقي — إضافة صفحات تسويقية تغيّر فكرة المنتج، لذا وُثّقت كتوصية فقط في IMPROVEMENT_PLAN.md ولم تُنفذ (التزاماً بشرط الحفاظ على الهوية).
