# Murad Learning Planner 🧠

## نظام التعلم اليومي المتقدم | Advanced Daily Learning System

منصة ويب احترافية لإدارة وتنفيذ نظام التعلم اليومي طويل المدى.

---

## ✅ الميزات المكتملة

### 🏠 Dashboard الرئيسي
- عرض الأسبوع الحالي، النمط، اليوم، الجلسة، البلوك، المؤقت، البلوك التالي، الحالة
- نظرة سريعة على الأسبوع (7 أيام)
- تقدم اليوم الحالي
- إحصائيات الأسبوع

### 📅 Weekly Pattern Board
- جدول أسبوعي قابل للتعديل الكامل
- تغيير عناوين الأعمدة مباشرة
- تعديل محتوى كل خلية مباشرة
- حفظ تلقائي
- التنقل بين الأسابيع
- فتح أي يوم مباشرة من الجدول

### 📖 Daily Execution Board
- بطاقة شرح النمط (A/B/C/D) بتفصيل كامل
- بلوكات زمنية قابلة للتعديل الكامل:
  - تعديل اسم البلوك
  - تعديل النشاط
  - إضافة/حذف بلوكات
  - إعادة الترتيب بالسحب والإفلات (Drag & Drop)
- نافذة تفاصيل البلوك (Side Panel) تحتوي على:
  - اسم البلوك، الوقت، النشاط، الهدف، ماذا سأتعلم، ماذا سأطبق، الملاحظات، المخرجات، نغمة التنبيه

### ⏱️ نظام المؤقت
- Start / Pause / Resume / Reset / Skip لكل بلوك
- شريط تقدم بصري
- عداد تنازلي
- تنبيه عند انتهاء الوقت
- Modal يعرض البلوك المنتهي والبلوك التالي
- إشعارات المتصفح

### 🔗 Integration Builder (النمط C فقط)
- إدخال المفاهيم الثلاثة (رياضيات، برمجة، علوم الحاسوب)
- مخطط بصري تفاعلي
- حقل شرح العلاقات

### 🔔 نظام الأصوات
- 10 نغمات مدمجة (مولدة بـ Web Audio API)
- معاينة كل نغمة قبل الاختيار
- رفع نغمات مخصصة (MP3, WAV, OGG, M4A)
- نغمة مختلفة لكل بلوك
- إعدادات متقدمة: مستوى الصوت، التكرار، المدة

### 📄 نظام PDF
- تصدير PDF يومي احترافي (صفحة غلاف + الجدول + الملاحظات + الخلاصة)
- تصدير PDF أسبوعي
- تصميم Premium مع هيدر وفوتر شخصي

### 🌐 ثنائية اللغة
- دعم كامل للعربية والإنجليزية
- تبديل فوري بين اللغتين
- حفظ تفضيل اللغة

### 🎨 التصميم
- Dark Mode / Light Mode
- Glassmorphism
- تصميم SaaS عالمي المستوى
- انميشن وتأثيرات حركية
- Tooltip احترافي على كل بلوك

### 💾 التخزين
- Local Storage مع بنية قابلة للتوسع
